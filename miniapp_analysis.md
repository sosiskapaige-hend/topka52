## FAQ — Полный текст (11 вопросов)

### Q1: Что такое Arbix?
Arbix — это автоматизированный арбитражный бот и агрегатор криптовалютных бирж, который помогает находить и использовать ценовые различия между торговыми площадками для извлечения прибыли.

Сервис анализирует доступные рынки, автоматически ищет арбитражные возможности (спреды) и помогает пользователю выполнять операции между биржами через API-интеграции.

Проект Arbix.

### Q2: Как работает Arbix?
Arbix подключается к биржам через API и анализирует цены на разных торговых площадках.

Когда система обнаруживает арбитражную возможность, она автоматически выполняет необходимые арбитражные операции через API.

Весь процесс происходит через интерфейс сервиса — пользователь взаимодействует с платформой, а операции выполняются автоматически.

### Q3: Получает ли Arbix доступ к моим средствам?
Нет. Администрация Arbix не имеет прямого доступа к средствам пользователей.

Сервис не хранит пользовательские активы и не управляет ими напрямую. Работа осуществляется через API-интеграции бирж и действия пользователя внутри интерфейса Arbix.

### Q4: Какие операции выполняет Arbix?
Сервис автоматизирует арбитражные операции между биржами, включая:

· поиск ценовых различий между площадками
· анализ спредов
· выполнение торговых операций через API
· автоматизацию процесса перевода и взаимодействия между биржами
· хранение криптовалютных активов

### Q5: Как Arbix зарабатывает?
Arbix получает доход за счёт комиссий за использование сервиса:

· комиссия за пополнение
· комиссия за вывод
· комиссия с каждой арбитражной операции

### Q6: Какой лимит операций доступен пользователю?
Без подписки пользователь получает до **100 операций** в сутки.

С подпиской Arbix+ лимит увеличивается до **300 операций** в сутки.

### Q7: Что такое Arbix+?
Arbix+ — это расширенная подписка стоимостью **89 USDT**, которая предоставляет дополнительные возможности.

Преимущества Arbix+:

· лимит до **300 операций** в сутки
· приоритетное зачисление пополнений
· приоритетная обработка операций
· отсутствие комиссии на вывод

### Q8: Какие комиссии есть в Arbix?
Стандартные условия:

· комиссия на пополнение — **0%**
· комиссия на вывод — применяется согласно условиям сервиса
· комиссия с каждой операции — **0,15%** от прибыли

Для пользователей Arbix+:

· приоритетное зачисление пополнений
· комиссия на вывод — **0%**
· приоритетная обработка операций

### Q9: Какие биржи поддерживает Arbix?
Arbix работает через API-интеграции с подключёнными биржами.

Список доступных площадок и поддерживаемых направлений отображается внутри сервиса и может расширяться.

### Q10: Нужно ли вручную искать арбитражные возможности?
Нет.

Одна из основных функций Arbix — автоматический поиск спредов между биржами.

Система анализирует рынок и помогает находить потенциальные возможности без необходимости постоянного ручного мониторинга.

### Q11: Нужны ли специальные технические знания?
Нет.

Arbix создан для автоматизации процесса и работает через понятный пользовательский интерфейс.

Пользователю не требуется самостоятельно выполнять технические операции.
